Models.PersistantObject module¶

class Models.PersistantObject.PersistantObject[Quellcode]¶

Bases: object

Klassen die von dieser Klasse erben sind Persistent und können ihre Einstellungen im Konfigurationsordner speichern. Diese Möglichkeit sollte von Singeltons wie Services benutzt werden.

Fügt in der obersten Ebene einen Mutterknoten für eine Einstellung hinzu.

Parameter:
  • key (str) – der Name des Mutterknotens der Einstellung
  • settings (dict) – die Einstellungen die unter diesem Muttknoten gespeichert werden sollen.
