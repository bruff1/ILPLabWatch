Models.Observable module¶

class Models.Observable.Observable[Quellcode]¶

Bases: object

Observable ist eine der zwei abstrakten Klassen die zur Implementation des Beobachtermusters notwendig ist. Erbt eine Klasse von Observable, so kann sie von Observern beobachtet werden.

addObserver(observer)[Quellcode]¶

Fügt dem Observable einen Beobacter hinzu. Dieser wird informiert, sobald sich am Observable etwas entscheidenes ändert.

Parameter:observer (Observer) – Das Beobachterobjekt welches beobachten soll..
informObserver()[Quellcode]¶

Diese Methode wird vom Kindobjekt aufgerufen, wenn sich etwas verändert hat. Es werden dann die Observer benachrichtigt.

informObserverOfType(type)[Quellcode]¶

Informiert alle Observer eines bestimmten Typs. Ist im klassischen Beobachtermuster nicht vorgesehen aber besonders in Pyton ziemlich nützlich

Parameter:type – der Typ der zu informierenden Observer
